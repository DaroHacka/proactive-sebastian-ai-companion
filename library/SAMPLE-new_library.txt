### SAMPLE: How to create a new library
###
### 1. Name your file: library-X-name.txt
###    - X = letter ID (d, e, f, etc.)
###    - name = descriptive name
###
### 2. Format: One entry per line
###    - Lines starting with # (one or more #) are IGNORED
###    - Each remaining line is a valid entry
###
### 3. Example for a "books" library:

[BOOK: THE_GREAT_GATSBY],"The Great Gatsby" by F. Scott Fitzgerald - Jazz age decadence.
[BOOK: 1984],"1984" by George Orwell - Dystopian surveillance state.
[BOOK: DUNE],"Dune" by Frank Herbert - Desert planet politics and spice.

###
### 4. To use custom loading logic:
###    - Create a Python function in a manager file
###    - Register in library_manager.py LIBRARIES dict
###    - OR just let the default loader pick random lines!
###
### 5. Auto-discovery rules:
###    - File must match: library-X-name.txt (X = single letter)
###    - Multiple files with same X → grouped as ONE library
###    - Delete file → library auto-removed on restart
