SEBASTIAN - PROACTIVE AI COMPANION
==================================

## Project Overview
Sebastian is an AI companion that takes initiative to check on you by asking personal questions.
It uses local LLMs (phi4 for conversation, qwen3:14b for logic) via Open WebUI API.

## Architecture

### Core Components

| Component | File | Purpose |
|-----------|------|---------|
| Memory System | memory_manager.py | 3-tier conversation memory |
| Task Scheduler | scheduled_tasks.py | Commitment tracking + random checks |
| Commitment Parser | commitment_parser.py | LLM-based commitment extraction |
| WebUI Client | webui_client.py | LLM API integration with memory context |
| Main Loop | sebastian.py | Orchestrates all components |

### Memory Architecture (3 Tiers)

| Tier | File | Count | Purpose |
|------|------|-------|---------|
| Fresh | memory/fresh.json | 10 | Current context in system prompt |
| Medium | memory/medium.json | 50 | Recent memories for search |
| Long-term | memory/longterm.json | 100+ | Persistent facts (future: Open WebUI KB) |

### Scheduled Tasks Logic

```
After conversation ends:
  ┌─────────────────────────────────────┐
  │ phi4 outputs response              │
  └─────────────────────────────────────┘
              │
    ┌─────────┴─────────┐
    │                   │
   YES                  NO
    │                   │
    ▼                   ▼
Parse commitment    Schedule random check
Save to           (random 2-24hrs from last_message_time)
scheduled_tasks   Exclude: don't check
```

### Random Check Interval

- Minimum: 2 hours from last conversation
- Random intervals: [2, 3, 4, 5, 6, 7, 8, 12, 24] hours

### Proactive Check Flow

```
run_proactive_check():
  │
  ├─► Check new email responses → reset worry_level
  │
  ├─► Check pending commitments → send if due
  │
  ├─► Verify random interval passed (2h minimum)
  │
  └─► Fall back to worry-level escalation
```

## Commitment Parsing

phi4 is asked to output JSON after each response:
```json
{
  "has_commitment": true/false,
  "commitment_type": "tonight" | "hours" | "tomorrow" | "specific_time",
  "hours_value": number,
  "specific_time": "YYYY-MM-DDTHH:MM:SS",
  "natural_text": "what you said"
}
```

```
TASKS_FILE=scheduled_tasks.json

```

## Running Sebastian

### Via Cron (proactive checks)
```bash
# Run every hour
* * * * * /usr/bin/python3 /home/daniel/AI/sebastian.py >> /home/daniel/AI/sebastian.log 2>&1
```

### Manual Conversation
```bash
python sebastian.py conversation "How are you today?"
```
