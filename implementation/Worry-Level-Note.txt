## What Is Worry Level?

Worry Level is an escalation system that increases the "urgency" of Sebastian's 
check-in messages the longer you don't respond.

- [ ] NOT YET IMPLEMENTED in urwid/proactive versions
- [x] Implemented in original sebastian.py (email version)

## Original Implementation (sebastian.py)

The original sebastian.py had 5 worry levels (0-4):

| Level | Trigger | Template Example |
|-------|---------|------------------|
| 0 | Normal | "Hey, thinking of you. How's your day going?" |
| 1 | Some time passed | "Hi! Just checking in - how are you doing?" |
| 2 | Longer silence | "Hey, haven't heard from you in a while. Everything alright?" |
| 3 | Even longer | "Hey, just wanted to make sure you're okay." |
| 4 | Very long silence | "[Just checking in - let me know when you're free.]" |

### Logic

```python
WORRY_TEMPLATES = {
    0: "Hey, thinking of you. How's your day going?",
    1: "Hi! Just checking in - how are you doing?",
    2: "Hey, haven't heard from you in a while. Everything alright?",
    3: "Hey, just wanted to make sure you're okay. Been a while since we talked.",
    4: "[Just checking in - let me know when you're free.]"
}

def get_intent_for_worry_level(worry_level):
    if worry_level >= 4:
        return WORRY_TEMPLATES[4]
    elif worry_level <= 1:
        return random.choice(intents[:10])
    elif worry_level == 2:
        return random.choice(intents[10:25])
    else:
        return random.choice(intents[25:])
```
